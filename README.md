# Player.js Iframe

[Player.js](https://github.com/embedly/player.js/) iframe adapters.

## Installation

```
$ npm install playerjs-iframe --save-dev
```

## Supported Adapters

- Vimeo
- YouTube

## Build

```
$ npm run build
```

## Development

```
$ npm run dev
```

Visit [http://localhost:8080](http://localhost:8080)

## Testing

```
$ npm run test
```
